## 🏃‍♀️ Getting Started

To run this portfolio locally:

```bash

git clone https://github.com/Maitry09/portfolio.git

# Navigate into the project directory
cd portfolio

# Open in browser
open index.html
```

No build tools or dependencies required — it's a static site!
